h2. Rizzoma Extended

Розширення додає поле для пошуку по сторінці на сервісі "Rizzoma":https://rizzoma.com.
* При відображенні результатів пошуку, блоки які не містять співпадінь — згортаються. Розгортаються при кліку.
* Гарячі клавиші: 
** / — перейти до поля пошуку
** Esc — очистити пошук

h3. Що не працює:
* Пошук працює лише для сторінки, яка відкривається на початку. Якщо перейти на іншу хвилю/топік — поле пошуку пропаде. Ще не знаю, чи зможу це виправити. Тож зараз щоб працював пошук на іншому документі потрібно на нього переключитися і оновити сторінку.
* Динамічно створений контент не індексується. Тобто якщо у процесі роботи було додано новий текст — він ще не буде знаходитися пошуком. Потрібно оновити сторінку щоб пошук працював для усіх текстів включаючи щойно додані.
